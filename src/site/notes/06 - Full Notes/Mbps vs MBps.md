# Mbps vs MBps
- 네트워크는 데이터를 전송할 때, 비트 단위로 전송한다.
- 따라서, 측정 단위가 비트인 **Mbps(Megabit Per Second)**다. 
- **MBps는 MegaByte Per Second**로 바이트 단위다.
