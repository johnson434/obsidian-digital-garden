# 오늘 학습 내용 백지 복습
- FSx (File System X)
- File System은 높은 가용성을 위해서 하나의 AZ에 복제본을 두는 방식과 멀티 AZ에 복제본을 두는 방식이 있다.
- AWS의 모든 저장소는 Storage Gateway를 통해서 온프로미스와 클라우드에 데이터를 같이 사용할 수 있다.
- Storage Gateway를 통해서 주기적인 백업이나 DR이 가능하다.
