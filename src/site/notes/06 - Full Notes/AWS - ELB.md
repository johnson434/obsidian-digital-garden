### ELB란?
- Elastic Load Balancer로 AWS에서 제공하는 로드 밸런서를 통칭하는 용어
- 다음 4가지가 존재한다.
	1. CLB(Classic Load Balancer): 구형 로드 밸런서로 7계층
	2. ALB(Application Load Balancer): 
	3. NLB(Netowk Load Balancer)
	4. GWLB(Gateway Load Balancer)
