# Providers란?
- 리소스 타입의 컬렉션을 제공하는 테라폼 플러그인으로 Terraform Registry로부터 다운 받는다.[^1]
- 각 리소스는 `provider`에 의해 구현된다.
- Go언어로 구현된 SDK로 직접 `provider` 개발 가능
